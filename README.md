# Homely

**Project Name:** Homely — A Scheduled Home-Based Marketplace Platform

## Overview
Homely brings customers and vendors closer by creating a scalable, slot-based marketplace for home-based businesses. This readme will guide the folder's purpose, setup, and progress tracking.

- **Customer WebApp:** UI/UX tailored for product discovery, purchasing, and order tracking.
- **Integrated Multi-User:** A single platform handling functionality for customers, vendors, admin, and delivery partners, while keeping things clear and user-friendly.

## Modules in Progress
1. **Customer:** Slot-based browsing, order placement.
2. **Vendor Management:** KYC uploading, inventory tracking.
3. **Admin Oversight:** Vendor analysis, commissions, and finances.
4. **Delivery Tools** (Optional Phase).

---

## Guidelines
- **UI/UX:** Strictly professional, elegant designs.
- **Tech:** React Native & Node.js, integrated database using PostgreSQL to track modular flows.
- "No Dark" means approachable & trustworthy design philosophy. Avoid heavy themes. Responsive always for compatibility.
